Helps you manage your downloads (restrict to users/roles).

== Description ==

Simple Downloads adds a new post type: Downloads. Use it to create custom pages for your downloads and restrict them to users/roles.

Plugin sponsored by [Vivanista.com](http://vivanista.com)
